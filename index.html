3D Scanning Atlas

Introduction

The University of California (UC) Libraries 3D Scanning Atlas is a collaboration among librarians and library staff at UC Santa Cruz, UC Davis, UC Riverside, UC Los Angeles, UC San Francisco, and UC Berkeley. All participating libraries are part of the University of California 3D Data Common Knowledge Group (CKG). The goal of the atlas is to create a collection of 3D scanned reference objects using different 3D scanning technologies available at UC Libraries.

The 3D Scanning Atlas provides qualitative, quantitative, and visual information to effectively evaluate how the 3D scanners available across the UC Library system interact with different types of materials and how they compare to one another. The objects chosen for 3D scanning represent various material properties important in the 3D scanning process. We hope the 3D Scanning Atlas enables campus constituents to make informed decisions about which 3D scanning option best fits their education and research needs and provides librarians and library staff with data to make informed purchases.

3D Scanners

The 3D Data Atlas includes a range of 3D scanners currently available across UC Libraries. The following criteria were identified for comparing the 3D scanners: resolution; ability to capture transparency/semi-transparency, cavities, and reflectivity; time to 3D scan; and price.

The following 3D scanners and digital cameras were used to use scan the objects at the respective libraries:
  • Artec Space Spider (UCLA)
  • Canon 6D (UCSC)
  • Einscan-SE (UCR)
  • Einscan-SP (UCSF)
  • Einscan Pro (UCB)
  • Einscan Pro 2X with HD attachment (UCSC)
  • Einscan Pro 2X with Color attachment (UCSC)
  • iPhone 13 (UCSF, UCSD, UCLA, UCSC)

Scanning Process

The 3D CKG members have a range of experience using 3D scanning technology. 3D CKG members agreed on the following process that would be beneficial for both novice and experts:
  • Use the 3D scanner manufacturer’s software to 3D scan whenever possible
  • Use default settings in 3D scanning software whenever possible
  • Follow manufacturer documentation rather than bringing in any personal knowledge about scanning
  • Use only lighting from environment and avoid using additional lighting
  • Time how long it takes to complete the 3D scanning process from start to finish
  • Export as both OBJ and STL whenever possible

The 3D CKG members acknowledge that there are a number of ways to improve the outcome of 3D scans (e.g., using additional lighting, powder coating, etc). We intentionally did not apply previous 3D scanning knowledge or techniques. Rather, we simply followed the manufacturers’ instructions. The intention is to produce 3D models for the Atlas that someone with minimal 3D scanning experience could expect.

Conclusion

  • Inexpensive 3D scanners can produce usable scans based on the characteristics of the object being scanned.
  • High-power, expensive 3D scanners tend to consistently produce better results.
  • University of California 3D Data Common Knowledge Group learned of the 3D scanning capabilities across UC Libraries. This can be helpful for collaboration and research within the UC Library system.

We hope the 3D Scanning Atlas enables campus constituents to make informed decisions about which 3D scanning option best fits their education and research needs and provides librarians and library staff with data to make informed purchases.
